ff4s - Feature Flags For Scala

Goals

  • Cross-platform, cross-version Scala feature flagging
  • Implementation of a Scala Flipt client
  • Implementation of a Scala OpenFeature SDK
  • Implementation of a Scala Flipt OpenFeature provider
  • Implementation of a Scala LaunchDarkly OpenFeature provider, e.g. with Catalyst
